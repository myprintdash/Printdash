Connect to Firebase — takes about 5 minutes, free forever
1
Create a free Firebase project
Go to console.firebase.google.com (sign in with Google) → Click "Add project" → Name it printdash → Disable Google Analytics → Click Create project.
2
Enable Firestore database
In your project → Left sidebar → Build → Firestore Database → Click "Create database" → Choose "Start in test mode" → Select any region → Click Enable.
3
Enable Email/Password login
Left sidebar → Build → Authentication → Click "Get started" → Click Email/Password → Toggle Enable → Save.
4
Get your Firebase config keys
Left sidebar → Project Settings (gear icon) → Scroll down → Under "Your apps" click the </> Web icon → Register app (any name) → Copy the config values below.
5
Create your admin account
